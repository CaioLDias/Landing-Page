# Landing-Page
Nesse repositório eu fiz uma Landing Page baseada no Pokémon Unite, que é um MOBA que gosto muito de jogar.

## Componentes
Para fazer esta Landing Page, utilizei de HTML5 e CSS3 (aplicando os conhecimentos de Flexbox).
